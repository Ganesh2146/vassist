⚙️ CUSTOMIZATION GUIDE
═══════════════════════════════════════════════════════════════════════════════════════

Change Primary Colors:
──────────────────────

Edit ultimate.css :root variables:

```css
:root {
  --neon-purple: #A855F7;    /* Change this */
  --neon-cyan: #06B6D4;       /* Or this */
  --neon-green: #10B981;      /* Or this */
  
  --primary-gradient: linear-gradient(
    135deg,
    #A855F7 0%,      /* Purple */
    #06B6D4 50%,     /* Cyan */
    #14B8A6 100%     /* Teal */
  );
}
```

Change Animation Speed:
───────────────────────

```css
:root {
  --duration-fast: 150ms;     /* Change this */
  --duration-normal: 300ms;   /* Or this */
  --duration-slow: 500ms;     /* Or this */
}
```

Change Blur Effects:
────────────────────

```css
:root {
  --blur-sm: blur(4px);
  --blur-md: blur(8px);
  --blur-lg: blur(16px);
}
```

Add New Button Style:
──────────────────────

